Goals

Technical demonstrator to show and test a 100% AJAX driven CRUD panel for virtual collections.
The focus is on functionality rather than layout. The CRUD panel contains two 1 to n editors:
(1) authors (inline editor) and (2) references (editor with background process).
References are analyzed in the background after adding. If possible type information and basic metadata is extracted.

Open issues

  • Improve reference editor so that it works inline (if possible).
  • Improve validation, including disabling final save button while the collection is invalid
  • Improve submission of inline editor. This currently is triggered by the last field. This should be made to submit when the whole inline editor lost focus. Example when filling in an author and tabbing "back" from the last (affiliation) field triggers a submit
  • Improve layout and responsiveness
  • Support editing of complete collections
  • Add confirmation before cancel or remove
